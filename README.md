# p2pcoin_bot

## 소개

가상화폐거래소 시세를 대략 매초마다 수신하여

해당 키워드에 대해 알람메세지를 보냅니다.

텔레그램 아이디 : p2pcoin_bot

## 사용법

* set

웹을 통해 알림메세지 조건을 설정합니다.

* % 통화 값

해당 통화(BTC, XMR등)의 등락율이 값 이상, 이하일 경우 알림메세지를 보내도록 조건을 등록합니다.
